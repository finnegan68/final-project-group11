Для удобства, предлагаю всем пользоваться GUI интерфейсом для Git. Скачать можно по ссылке: https://desktop.github.com/

После установки, необходимо запустить Git Shell с рабочего стола (если создался ярлык, если нет, то с меню Пуск). Для того, чтобы не искать репозитории в файловой системе, создайте в корне любого из дисков папку GitHub (если её ещё нет). После создание необходимо в Git Shell перейти в эту директорию (команда cd “путь¬_к_директории”) и выполнить следующую команду: git clone https://github.com/Captious/final-project-group11 (Это общий репозиторий, в котором будет выполняться финальный проект). 

После того, как Git загрузит проект на компьютер, необходимо запустить GitHub по аналогии с Git Shell. Тут необходимо добавить локальный репозиторий. Делается это нажатием на «+» в левом верхнем углу. Во вкладке «Add» необходимо указать путь к папке, которая создалась командой git clone. 

Чтобы открыть проект в IntelliJ IDEA, необходимо нажать на File -> Open… и указать путь к папке «FinalProject». Можно приступать к выполнению своей задачи.

Чтобы закоммитить свои изменения необходимо открыть программу GitHub и открыть в ней свой локальный репозиторий. Перед коммитом обязательно делать синхронизацию и билд проекта (во избежание ошибок). Синхронизация делается с помощью нажатия на кнопку «Sync» в правом верхнем углу в GitHub. В левой колонке открывшейся вьюхи можно наблюдать измененные файлы и выбрать те, которые надо закоммитить, а в правой те изменения, которые были внесены. Обязательно для заполнения поле Summary при коммите. В нем надо кратко описать то, что было сделано, желательно на английском.

P.S.: Билд делается в IDE. Для этого в правой стороне необходимо выбрать вкладку «Maven Projects» открыть вкладку «Lifecycle» и выбрать две задачи: clean и install. После этого нажать на кнопку «Run Maven Build» (треугольник вверху вьюхи).
